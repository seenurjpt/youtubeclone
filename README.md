## Run Locally

Clone the project

```bash
  git clone https://github.com/seenurjpt/youtubeclone
```

Go to the project directory

```bash
  cd youtubecone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
